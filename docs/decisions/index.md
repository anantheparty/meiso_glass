# Decision Records

本目录记录会影响长期 SDK contract 的设计决策。它不是第二个 bible；唯一 bible 仍然是 `docs/SDK/bible/SDK_DESIGN_OVERVIEW.md`。

记录规则：

- 只记录会影响 wire、runtime、ABI、profile、policy 或跨语言实现的决定。
- 每条记录必须说明采用、借鉴或避免什么。
- 重要外部协议或文档必须留下链接。
- 未定问题直接写 `Open Questions`，不要伪装成已定规范。
